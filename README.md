# Arize - Your Personalized Gym Companion
Arize is a modern, Jetpack Compose-based Android application designed to help you manage your fitness journey. Whether you're tracking your gym sessions, monitoring your diet, or staying connected with a fitness community.

## Features

- **Gym Landing Page**: Set your workout schedule by selecting specific days and setting reminder notifications.
- **5-Tab Navigation**:
    - **Gym**: Your central hub for workout planning.
    - **Food**: Track your daily meals and nutrition (Coming Soon).
    - **Social**: Connect with friends and share your progress (Coming Soon).
    - **Stats**: Visualize your fitness data and achievements (Coming Soon).
    - **Profile**: Manage your personal info and settings (Coming Soon).
- **Modern UI**: Built with Material 3 for a sleek and intuitive experience.

## Getting Started

### Prerequisites

- Android Studio (Ladybug or newer)
- Android SDK 26+
- Kotlin 2.0+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Nl-T-lN/Arize.git
   ```
2. Open the project in Android Studio.
3. Sync the Gradle project.
4. Run the app on an emulator or physical device.

## Technologies Used

- **Jetpack Compose**: For the modern, declarative UI.
- **Material 3**: For consistent and beautiful design elements.
- **Navigation Suite**: For adaptive and easy-to-use navigation.
